如何用hugo搭建博客

如何用hugo搭建博客 第一步 去hugo release 页面下载 hugo_xxx_Windows-64bit.zip 解压hugo并将其配置到path 重启终端，运行hugo version 查看版本 第二步 进入hugo官网，点击QuickStart快速开始 从Step2 开始复制粘贴代码到终端，直到Step7 查看hugo版本 hugo version 创建新的网站 hugo new site quickstart 创建默认主题 git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke echo 'theme = "ananke"' >> config.toml 创建第一篇博客 hugo new posts/my-first-post.md 预览草稿 hugo server -D 打开 config.toml 进行博客配置 baseURL = "https://example.org/" languageCode = "en-us" title = "My New Hugo Site" theme = "ananke" 得到一个public目录，这是博客站点
